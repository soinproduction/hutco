Для разработчиков

Подключение сайта к ЕРИП осуществляется через специальное API, разработанное сервисом «Хуткi Грош»™, либо через один из модулей CMS (той, что у Вас установлена). Благодаря этому Вы можете формировать счета и получать оплату за товары и услуги в реальном времени в любой точке мира.

Подключение сайта через API

Чтобы подключить Ваш сайт через API, воспользуйтесь документацией здесь. Вы можете сделать это сами или передать информацию разработчикам. Для интеграции на языке php можете воспользоваться библиотекой cmsgate-hutkigrosh-lib

Подключение сайта через модуль CMS
Чтобы подключить Ваш сайт к ЕРИП, мы предоставляем специальные модули для следующих CMS-систем:
  • Joomla Joomshopping
  • Joomla Virtuemart
  • OpenCart:
  • v1.5
  • v2.1
  • v2.2
  • v2.3
  • v3.0
  • CRM Bitrix 24 (коробочная версия)
  • Bitrix. Управление сайтом
  • WordPress Woocommerce
  • CS-Cart
  • Drupal:
  • v7.x
  • v8.x
  • Modx Minishop2
  • Prestashop
